## 🔢 Array
> **Strategic Focus:** Foundation of DSA — learn core traversal, searching, and subarray techniques.

- **Sliding Window**
  - Fixed Size
  - Variable Size (Expand-Shrink, Monotonic Window)
- **Two Pointer**
  - Opposite ends
  - Fast & slow pointers
  - Partition / Dutch flag
- **Prefix Based**
  - Prefix Sum / XOR / 2D Prefix
- **Kadane’s / Subarray**
  - Max sum / product subarray
  - Subarray with given XOR / sum
- **Binary Search**
  - On index
  - On answer

---

## 🔤 String
> **Strategic Focus:** Pattern recognition and manipulation.

- **Sliding Window**
  - Longest substring without repeat
  - Minimum window substring
  - Anagram / permutation in string
- **Two Pointers**
  - Palindrome check
  - Reverse words / characters
  - String compression
- **Pattern Matching**
  - KMP
  - Rabin-Karp
  - Z-algorithm

---

## 🗂️ Hash Map
> **Strategic Focus:** Fast lookups, frequency analysis, and grouping.

- Frequency Based
- Lookup Based
- Set Based
- Index Mapping
- Grouping Pattern

---

## 📚 Stack
> **Strategic Focus:** Monotonic structures, nearest element queries, and expression evaluation.

- Monotonic Stack (Increasing / Decreasing)
- Nearest Element (Next Greater / Smaller / Previous)
- Range / Span
- Min/Max Stack
- Expression Handling
- Histogram Pattern

---

## 📬 Queue / Deque
> **Strategic Focus:** FIFO processing and sliding window optimizations.

- FIFO Processing
- Level-wise Processing
- Circular Queue Pattern
- Deque Based

---

## 🔗 Linked List
> **Strategic Focus:** Pointer manipulation and structural transformations.

- Fast-Slow pointers / Cycle detection
- Reversal (Full / k-group)
- Merge Lists

---

## 🌳 Trees
> **Strategic Focus:** Hierarchical data, traversal strategies, and path-based problems.

- Traversal (DFS / BFS)
- Recursion Patterns (Top Down / Bottom Up)
- Path Based (Max path sum, Diameter, Height)
- Binary Search Tree (BST)

---

## 🔄 Recursion
> **Strategic Focus:** Backtracking and divide & conquer paradigms.

- Backtracking
  - Decision Tree
  - Subsets / Permutations / Combinations
  - Word search / Palindrome partitioning
  - Pruning / State Tracking
- Divide & Conquer
  - Merge Sort
  - Quick Select
  - Count Inversions

---

## ⚖️ Heap
> **Strategic Focus:** Priority-based processing and greedy optimizations.

- Top K / Kth Element
- Greedy + Heap (Task scheduler, Meeting rooms, Huffman encoding)
- K-way Merge

---

## 🌐 Graphs
> **Strategic Focus:** Traversals, shortest paths, and connectivity.

- BFS / DFS
- Cycle Detection (Directed / Undirected)
- Topological Sort (Kahn’s / DFS)
- Shortest Path (Dijkstra / Bellman-Ford / Floyd-Warshall)
- Spanning Tree (Kruskal / Prim’s)
- Union-Find (DSU)
- Bipartite / Multi-source BFS / 0-1 BFS

---

## 🔎 Trie
> **Strategic Focus:** Efficient prefix-based searching.

- Insert / Search
- Prefix Match
- Bitwise Trie

---

## 🧮 Dynamic Programming
> **Strategic Focus:** Optimal substructure and overlapping subproblems.

- Core (1D / 2D)
- Transition Types (Linear / Grid / Decision DP)
- Pattern Types (Knapsack / Sequence / Partition / Interval)
- Advanced (Bitmask / Digit / DP on Trees)
- Optimization (Memoization / Tabulation)

---

## 💡 Greedy
> **Strategic Focus:** Local optimal choices leading to global solutions.

- Interval Greedy (Activity Selection, Non-overlapping Intervals)
- Scheduling Greedy (Deadline / Profit Based)
- Resource Allocation (Platforms / Meeting Rooms)
- Jump Game Pattern
- Huffman / Merge Cost

---

## ⚙️ Bit Manipulation
> **Strategic Focus:** Low-level optimization and subset generation.

- XOR Pattern
- Bit Masking
- Subset via Bits
- Bit Checks
- Prefix XOR

---

## 🔄 Sorting Algorithms
> **Strategic Focus:** Foundational sorting techniques.

- Bubble / Selection / Insertion Sort
- Merge / Quick Sort
- Heap Sort
- Counting / Radix / Bucket Sort

---

## 📏 Range Structures
> **Strategic Focus:** Efficient range queries.

- Segment Tree (Range Query / Lazy Propagation)
- Fenwick Tree (Prefix Query)

---

## 🎯 Strategic Learning Path
1. **Foundation:** Arrays, Strings, Hash Maps, Linked Lists  
2. **Intermediate:** Stacks, Queues, Trees, Recursion  
3. **Advanced:** Graphs, Heaps, Tries, Dynamic Programming  
4. **Optimization:** Greedy, Bit Manipulation, Range Structures
